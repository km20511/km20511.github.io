동아리 활동

  현재는 다음과 같은 활동들을 기획하고 있습니다. 활동 중 일부는 할 수도, 하지 않을 수도 있습니다. 신청 시 하고 싶은 활동을 알려 주시기 바랍니다.

1.Python 텍스트 프로그래밍

input과 print만 사용하는 간단한 입출력으로 실생활에 쓸 수 있는 간단한 프로그램이나 텍스트 게임 등을 만들어 봅니다.

2.Python & Pygame 2D 게임

Python에서 사용할 수 있는 2D 게임 엔진인 Pygame을 사용하며 첫 GUI(Graphic User Interface) 프로그래밍을 해 보고 게임이 작동하는 원리를 배웁니다.

3.Unity 3D 시뮬레이션

가장 많은 사람들이 사용하는 게임 엔진 중 하나인 Unity를 이용해 2D 또는 3D 게임을 만들어 보며 컴포넌트와 같은 게임 프로그램에서 많이 사용되는 프로그래밍 패턴과 새로운 언어 C#에 대해 배웁니다.

4.HTML & CSS 웹페이지

우리가 보는 모든 웹페이지들을 구성하는 언어 HTML에 대해 알아보고 웹페이지를 만드는 방법을 공부해 봅니다.

5.알고리즘 기초 배우기

프로그래밍으로 여러 가지 문제를 해결하고 프로그램을 더 효율적으로 작성하기 위한 알고리즘의 기초에 대해 배우고 알고리즘 문제를 풀어 보는 활동을 합니다.

6.인공지능 & 기계학습 등 신기술 알아보기

현재 여러 분야에 사용되며 트랜드가 되고 있는 인공지능과 같은 기술들의 기초적인 원리에 대해서 탐구하고 알아보는 활동을 합니다.
